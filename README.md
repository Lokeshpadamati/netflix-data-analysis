## 📌 Project Overview
This project analyzes Netflix data to uncover trends in content, genres, and growth over time.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## 📊 Key Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Interactive dashboard in Power BI

## 📈 Key Insights
- Movies dominate Netflix content
- USA produces the most content
- Content increased rapidly after 2015
- Drama is the most popular genre

## 📁 Files
- notebook.ipynb → Python analysis
- cleaned_netflix_data.csv → dataset
- dashboard.pbix → Power BI dashboard

## 🚀 Outcome
Built an end-to-end data analysis project from raw data to dashboard.
